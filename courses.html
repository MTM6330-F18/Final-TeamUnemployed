Web Production

MTM6330

Use industry standard developer tools like git, GitHub, Node.js and Gulp to complete your work more quickly and consistently. Learn to employ CSS best practices and leverage pre-processors like Sass to write CSS code that is easier to read and maintain.

Server-Side Web Development

MTM6331

Throughout the course you will learn the PHP Scripting Language, the most widely used server-side development language, and MySQL, one of the most widely used, open source relational database management systems. Together you will be able to build dynamic web sites, securely track users interactions, and even create your own APIs.

Motion Graphics

MTM6300

Learning digital footage or animation which create the illusion of motion or rotation, and are usually combined with audio for use in multimedia projects. Created using Adobe After Effects

Project Management

MTM6361

Effective management of time, resources, clients, and teams is crucial to the success of any project. Students learn the theory of project management, following industry standard guidelines and best practices. The full project cycle is explored, including planning, managing risks, intellectual property rights, client and team communication, project scope, project documentation and closure.

Applied Projects

MTM6360

Students are provided an opportunity to work with real clients or on industry-reflective projects, using their skills to build real-world applications and multimedia products. Working in teams, students plan, manage and develop a project through the full project cycle.

General Electives

Many

Students choose one course from a group of general education electives, which meets one of the following theme requirements: Arts in Society, Civic Life, Social and Cultural Understanding, and Science and Technology.
